# Code of Conduct

This project adopts the
[Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, be constructive, assume good faith, and keep
disagreements technical. Harassment, personal attacks, and sustained
disruption are not tolerated.

## Enforcement

Report conduct issues privately to the maintainer (see repository owner
profile). Reports are handled confidentially. The maintainer may warn,
moderate, or ban at their discretion, following the Contributor
Covenant's enforcement guidelines.
