# What an agent should be allowed to do to my notes

Read everything, write nothing — directly. The asymmetry is the whole
design. An agent with retrieval over my corpus is uncontroversially
useful: it cites what I actually wrote, finds what I forgot, and
grounds its answers in my own words.

An agent with a pen is a different animal. Notes are a long-lived
asset with no undo across years; a helpful rewrite that subtly garbles
one date costs more than a hundred good summaries earn. So the write
path must be a proposal: the agent stages a diff, a human reads it,
and only a human applies it. Boring, auditable, reversible — exactly
what you want touching the canonical store.
